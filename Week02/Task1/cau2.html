Câu 2

  • Responsive là phong cách thiết kế website phù hợp trên tất cả các thiết bị, mọi độ phân giải màn hình.
  • Vô hiệu hóa tính năng Responsive:
    1. Bỏ thẻ <meta> viewport
    2. Ghi đè thuộc tính width trên .container đối với mỗi cột trong hệ thống lưới, ví dụ width: 970px !important;
    3. Vô hiệu hóa toàn bộ các tính năng thu gọn và mở rộng trên thanh điều hướng.
    4. Bố cục dạng lưới, sử dụng class .col-xs-* bằng cách thêm vào hoặc thay thế cho các class medium/large.
